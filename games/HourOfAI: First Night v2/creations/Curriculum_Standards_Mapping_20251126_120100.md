## **1. EXECUTIVE SUMMARY**

### **Overview of Standards Alignment**
*Hour of AI: The First Night* integrates **coding, artificial intelligence literacy, and problem-solving** within a narrative-driven Minecraft survival scenario.  
While its core focus is **computer science**, the experience naturally supports **Common Core English Language Arts (ELA)** and **Mathematics Practice Standards** through reading technical instructions, reasoning quantitatively, and communicating findings.

### **Key Learning Domains Covered**
| Domain | Description | Example in Game |
|--------|--------------|----------------|
| **Reading Informational Text (ELA)** | Interpreting on-screen guides, NPC dialogue, and coding instructions | Reading the in-game “Player’s Guide” and AI mentor prompts |
| **Writing and Communication (ELA)** | Explaining code logic, documenting AI training results | Writing reflections or explanations of AI behavior |
| **Mathematical Reasoning (Math Practices)** | Applying logic, sequencing, and conditional reasoning | Coding “if–then” statements for AI Agent behavior |
| **Problem Solving and Modeling (Math)** | Using computational models to represent real-world scenarios | Modeling AI classification and resource optimization |
| **Critical Thinking and Collaboration** | Engaging in team-based debugging or AI design | Group reflection on AI ethics and accuracy |

### **Grade Level Recommendations**
| Grade Band | Focus | Common Core Integration |
|-------------|--------|--------------------------|
| **3–5 (Upper Elementary)** | Intro to algorithms, reading comprehension of procedural text, logical reasoning | CCSS.ELA-LITERACY.RI.3–5; CCSS.MATH.PRACTICE.MP1–MP4 |
| **6–8 (Middle School)** | Data interpretation, problem decomposition, explanation writing | CCSS.ELA-LITERACY.W.6–8; CCSS.MATH.CONTENT.6.EE.B.6 |
| **9–12 (High School)** | Evaluating AI systems, argumentation, modeling real-world systems | CCSS.ELA-LITERACY.RST.9–10.3; CCSS.MATH.PRACTICE.MP3–MP5 |

## **6. ASSESSMENT ALIGNMENT**

### **Formative Assessments**
- **Observation Checklist:**  
  - Follows procedural steps accurately (RST.6-8.3)  
  - Demonstrates persistence in debugging (MP1)  
  - Communicates reasoning clearly (SL.5.1)
- **In-Game Evidence:**  
  - AI Agent performs coded actions correctly  
  - Player revises code based on feedback  
  - Player completes all survival tasks

### **Summative Assessments**
- **Reflection Journal or Presentation:**  
  - Explain how AI learns and improves (W.6.2, SL.8.4)  
  - Describe coding logic used and its outcomes (MP2, MP4)
- **Student Artifact Portfolio:**  
  - Screenshots of code  
  - Flowcharts or pseudocode diagrams  
  - Written summary of AI performance and ethical reflection
