## **1. EXECUTIVE SUMMARY**

### **Overview of Standards Alignment**
*Hour of AI: The First Night* aligns directly with the **Common Curriculum Framework for K–12 Computer Science Education**, emphasizing **computational thinking**, **computing technology**, and **societal impacts of computing**. The game introduces foundational **AI literacy**, **coding**, and **digital citizenship** concepts through interactive, inquiry-based gameplay.

**Core Domains Addressed:**
| Framework Domain | Description | Game Integration |
|------------------|-------------|------------------|
| **Computational Thinking (CT)** | Problem decomposition, pattern recognition, abstraction, algorithmic reasoning | Coding challenges, debugging tasks, AI training |
| **Computing and Networks (CN)** | Understanding how computing systems and AI agents function | Code Builder interface, AI agent behavior |
| **Data and Analysis (DA)** | Collecting, classifying, and interpreting data | AI classification tasks, verification exercises |
| **Impacts of Computing (IC)** | Ethical and societal implications of AI | Reflection prompts on human oversight, AI bias |
| **Design and Development (DD)** | Designing and testing computational solutions | Coding the AI agent, iterative debugging cycles |

### **Grade Level Recommendations**
- **Primary Focus:** Grades 4–8 (Ages 9–14)  
- **Adaptable For:** Grades 9–10 (introductory computer science or digital literacy courses)  
- **Learning Level:** Introductory to intermediate coding and AI concepts

## **2. DETAILED STANDARDS MAPPING**

### **Framework:** Common Curriculum Framework for K–12 Computer Science Education (Canada)

| **Standard Code** | **Standard Description** | **How the Game Addresses the Standard** | **Evidence from Gameplay** | **Depth of Coverage** | **Assessment Opportunities** |
|--------------------|--------------------------|-----------------------------------------|-----------------------------|-----------------------|-------------------------------|
| **CT1.1** | Decompose problems into smaller, manageable parts. | Students break survival goals (e.g., building shelter) into coding tasks (collect, craft, build). | Task sequence: Collect wood → Craft planks → Build shelter. | Reinforcement | Observation of student code structure and task planning. |
| **CT1.2** | Identify and apply patterns to solve computational problems. | Pattern recognition used in crafting recipes and AI classification. | AI learns patterns for identifying wood types and crafting tools. | Reinforcement | Students explain AI pattern recognition in reflection. |
| **CT1.3** | Design and test algorithms to solve problems. | Students design code for the AI agent to execute tasks and test/debug results. | Code Builder exercises and agent testing. | Mastery | Code accuracy and debugging logs. |
| **DA2.1** | Collect and organize data for computational use. | Players label and categorize materials (e.g., “wood” vs. “stone”) for AI training. | AI classification activity in Task 1. | Introductory | Student labeling accuracy and AI performance. |
| **DA2.2** | Analyze data to identify trends and inform decisions. | Students evaluate AI accuracy and adjust parameters. | Reflection on agent errors and corrections. | Reinforcement | Reflection journal: “How did the AI improve?” |
| **DD3.1** | Develop programs using sequences, loops, and conditionals. | Coding tasks require logical structures (if/then, loops). | Agent mining logic: “If stone, mine; if lava, stop.” | Mastery | Review of code blocks or Python scripts. |
| **DD3.2** | Debug and refine computational solutions. | Players identify and correct AI misclassifications or logic errors. | Agent fails to collect correct blocks → students debug code. | Mastery | Observation checklist or peer review. |
| **IC4.1** | Discuss the impact of computing technologies on society. | In-game prompts and post-lesson reflections explore AI ethics and responsibility. | Discussion: “Should the AI act on its own?” | Reinforcement | Reflection discussion or written response. |
| **IC4.2** | Demonstrate responsible use of technology and data. | Students practice ethical oversight of AI decisions and data labeling. | AI mentor dialogue on human-in-the-loop responsibility. | Reinforcement | Student self-assessment on ethical decision-making. |
| **CN5.1** | Explain how computing systems and devices work together. | Students observe how the AI agent, player, and code interface interact. | Agent executes player code via Code Builder. | Introductory | Teacher questioning during gameplay. |
| **DD3.3** | Collaborate to create computational solutions. | Students work in pairs or teams to code and test AI behaviors. | Team-based shelter building or AI optimization challenge. | Reinforcement | Peer collaboration rubric. |
